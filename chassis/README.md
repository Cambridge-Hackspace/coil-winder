# coil-winder : chassis

This is where I'll keep notes on the chassis. Most of this will 3D print
designs, and eventually stuff for the CNC machine.

## Technologies

I'll be using native OpenSCAD. It's possible I may pivot to something like
Python with an OpenSCAD library (especially if I decide I want animated
videos or something) but for now, OpenSCAD will do the job.

I make use of a few OpenSCAD libraries that you'll need if you want to
replicate any of this work. (I'll do my best to keep this list up to date).

- [nmasse-itix/OpenSCAD-Pattern-Filling](https://github.com/nmasse-itix/OpenSCAD-Pattern-Filling)
- [rcolyer/threads-scad](https://github.com/rcolyer/threads-scad)

I typically use the OpenSCAD nightly build and I am much too lazy to update
this repo each time I pull down a new build. So... you can take a look at the
commit dates for this repo and assume that I'm not using a nightly build
released after those dates :)

For printing, I'm mostly using the Snapmaker U1. I've been continually
impressed by this machine, but that may be because I'm used to working with my
old Anycubic Kobra Neo--which I have a lot of nostalgia for, but let's face
it... sometimes you just want the thing to work instantly.

I'm primarily using PLA for this build, at least for the prototype. I may switch
over to PETG or something later, I haven't decided. In particular, I'm very
tempted to try to print the threaded rod (which is probably a dumb idea, but
I'm intrigued), and if I do that I will almost certainly use either PETG or a
carbon-infused filament. (If I do end up using carbon, I'll begrudgingly pivot
to a Bambu X1 for that part.)
